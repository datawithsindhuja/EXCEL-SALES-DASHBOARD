&#x20;**Pizza Sales Dashboard (Excel)**



&#x20;About the Project

*T*his project is an Excel-based Pizza Sales Dashboard created to analyze pizza sales data and provide meaningful business insights. The dashboard uses Pivot Tables, Pivot Charts, and Slicers to visualize important metrics such as total revenue, total orders, pizzas sold, sales by category, sales by size, and top-selling products. It helps identify sales trends and customer purchasing patterns in an easy-to-understand format. This project demonstrates basic Excel data analysis, data visualization, and dashboard creation skills, making it a useful portfolio project for beginners interested in data analytics and business analysis.



Tools Used

\- Microsoft Excel

\- Pivot Tables

\- Pivot Charts
